About

Hi there! I'm Bianca, a Data Scientist, and slighty obsessed with teaching anything related to RStudio. I realised that there was a great need for tailored training programmes that teaches foundational computational skills, and I decided to start my own company, Conquest Analytics and Training. If you have a project that you want to get started or would like to get training in data analysis, visualization and report writing using RStudio, then get in touch.
